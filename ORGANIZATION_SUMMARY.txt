KEY FEATURES & KEYBINDINGS:
================================================================================

WINDOW NAVIGATION:
  Ctrl+H  - Focus left window
  Ctrl+J  - Focus lower window
  Ctrl+K  - Focus upper window
  Ctrl+L  - Focus right window

COMMENT TOGGLING (NEW!):
  Leader + /  - Toggle comment (any language)
  Works in both normal and visual modes

FILE EXPLORER:
  Leader + E  - Toggle Neo-tree

FIND/SEARCH:
  Leader + FF - Find files
  Leader + FG - Grep text
  Leader + UT - Change colorscheme

FORMATTING:
  Leader + P  - Format file or selection

GIT COMMANDS (NEW!):
  Leader + GS - Git status
  Leader + GD - Git diff
  Leader + GC - Git commit
  Leader + GB - Git blame
  Leader + GL - Git log
  Leader + GP - Git push
  Leader + GU - Git pull

GIT SIGNS (NEW!):
  Leader + GHB - Blame line
  Leader + GHD - Diff this
  Leader + GHH - Preview hunk

GITHUB COPILOT (NEW!):
  Leader + COP - Toggle Copilot
  Tab - Accept suggestion
  Alt+] / Alt+[ - Next/previous suggestion

COPILOT CHAT (NEW!):
  Leader + CCH - Open chat
  Leader + CCE - Explain code
  Leader + CCR - Review code
  Leader + CCF - Fix/debug code

BUFFER NAVIGATION:
  Shift + H - Previous buffer
  Shift + L - Next buffer

QUICK START:
================================================================================

1. Start Neovim:

2. Wait for plugins to install (1-2 minutes on first run)

3. Setup Copilot (optional):
   :Copilot auth

4. Start using new features:
   - Comment: Leader + /
   - Git: Leader + G*
   - Chat: Leader + CC*

================================================================================
ADDING NEW PLUGINS:
================================================================================

1. Create file in appropriate category:
   ~/.config/nvim/lua/plugins/[category]/plugin-name.lua

2. Add to lua/plugins/init.lua:
   require("plugins.[category].plugin-name"),

3. Open Neovim - plugin will auto-install
